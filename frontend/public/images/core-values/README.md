# Core Values Images

This directory contains background images for the Core Values section.

## Required Images:

1. **professionalism.jpg** - Business professionals in meeting/collaborative setting
2. **work-hard-play-hard.jpg** - Team celebration or modern office social environment
3. **open-communication.jpg** - Team discussion or brainstorming session
4. **grow-together.jpg** - Professional development or mentoring scenario
5. **team-before-me.jpg** - Collaborative team project or group success moment

## Image Specifications:
- Format: JPEG or WebP (WebP preferred for performance)
- Dimensions: 1920x1080 or similar 16:9 aspect ratio
- Quality: High resolution but optimized for web
- File size: Under 500KB each
- Style: Professional, modern, business-related

## Usage:
These images are used as background images in the CoreValues.jsx component with dark overlays for text readability.
